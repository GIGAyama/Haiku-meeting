# 🌸 GIGA句会プラザ 〜こころを詠む〜

**「明日の授業を変える、最強のGIGAスクールインフラ」**

GIGA句会プラザは、プログラミング知識ゼロの先生でも**「たった3分のコピペ」**で導入できる、完全無料の次世代句会Webアプリケーションです。

国語の俳句づくりはもちろん、理科の実験後や体育の試合後など、すべての授業の「振り返り」を五・七・五で行うことで、文章を書くのが苦手な児童の自己肯定感を高め、クラス全員の学びを美しく共有します。

*(ここにアプリのスクリーンショット画像を1〜2枚入れると効果的です)*

## ✨ 圧倒的な特徴（神アプリたる所以）

### 1. 究極の「Zero-Config」導入

先生が面倒なスプレッドシートの新規作成や、IDの紐付け設定を行う必要は一切ありません。コードを貼り付けてデプロイするだけで、**初回起動時に専用のデータベースがあなたのGoogleドライブ内に全自動で構築**されます。

### 2. 商用SaaSレベルのUI/UX

画面遷移のないReactによるSPA（シングルページアプリケーション）構成。すべてのボタンは子供の指でもタップしやすいサイズに調整され、美しい角丸、グラスモーフィズム、和風フォントを採用。大企業のアプリに匹敵する「触っていて気持ちいい」体験を提供します。

### 3. 全ての漢字に完璧なルビ（ふりがな）

低学年の児童でも一人で操作できるよう、画面上のすべての主要テキストにルビが振られています。

### 4. 先生を救う「最強のセーフティネット」

先生用ダッシュボードには、参加人数やコメント数をリアルタイムで表示するグラフを搭載。さらに、万が一の不適切コンテンツも「👁️ 隠す」ボタンをワンタップするだけで、広場から即座にミュート（非表示）にできます。

## 📱 主な機能

### 👧 児童向け機能

*   **リアルタイム短冊プレビュー:** 文字を打ち込むと、目の前で「本物の短冊」が出来上がっていく感動のUI。
    
*   **文字数ナビゲーション:** 「現在○文字（目安:5）」と優しく教えてくれるサポート機能。
    
*   **紙吹雪（Confetti）アニメーション:** 投票完了時や結果発表の瞬間、画面いっぱいに紙吹雪が舞う最高の演出。
    
*   **作品広場（スマートリロード）:** 画面を更新せずにワンタップで最新の作品を読み込み。新着順・ランダム順の並び替えも可能。
    
*   **マイページ:** 自分が過去に詠んだすべての作品と、もらったコメントを一覧で振り返り。
    

### 👨‍🏫 先生向け機能（管理ダッシュボード）

*   **リアルタイム統計グラフ:** 参加人数、投稿数、投票数などを美しく可視化。
    
*   **お題・投票ステータスの管理:** 「受付中」「締切（結果発表）」をワンタップで切り替え。
    
*   **ワンタップ・ミュート:** 不適切な作品を児童の画面から非表示にする安全装置。
    
*   **ワンクリックリセット:** 現在の句会データを過去のアーカイブとして自動保存し、次の句会を即座に開始。
    
*   **CSVエクスポート:** 句会の結果を成績処理用に1クリックでダウンロード。
    

## 🚀 導入手順（たったの3ステップ！）

事前の準備は「Googleアカウント」だけ。サーバー代などは一切かかりません。

1.  **プロジェクトの作成**
    
    *   Google Apps Scriptのエディタ（ [https://script.new/](https://script.new/ "null") ）を開きます。
        
    *   もともと入力されているコードをすべて消し、このリポジトリの `コード.gs` の内容を貼り付けます。
        
2.  **フロントエンドの作成**
    
    *   左側の「ファイル」横の「＋」ボタンを押し、「HTML」を選択します。
        
    *   ファイル名を `index` にし、このリポジトリの `index.html` の内容をすべて貼り付けます。
        
3.  **デプロイ（公開）**
    
    *   右上の青い「デプロイ」ボタン ＞「新しいデプロイ」をクリックします。
        
    *   左上の歯車マークから「ウェブアプリ」を選択します。
        
    *   以下の設定にして「デプロイ」を押します。
        
        *   実行するユーザー: `自分`
            
        *   アクセスできるユーザー: `全員`（またはご自身の学校の環境に合わせて設定）
            
    *   **表示されたURLを児童に共有するだけで、準備完了です！**
        

> **💡 ヒント:** > 初めて先生がアクセスした瞬間、裏側でスプレッドシート「【自動生成】GIGA句会プラザ_DB」が自動作成されます。 管理ページへの初期パスワードは `1234` です。（ログイン後、必ず変更してください）

## 🛠 技術スタック

アプリの堅牢性とカスタマイズ性を両立するため、モダンな技術をCDN経由で採用しています。

*   **Backend:** Google Apps Script (GAS)
    
*   **Database:** Google Spreadsheet
    
*   **Frontend:** React 18, Babel (CDN)
    
*   **Styling:** Tailwind CSS (CDN)
    
*   **Animation:** canvas-confetti
    
*   **Icons:** SVG (Inline)
